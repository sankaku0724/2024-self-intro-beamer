# 2024-self-intro-beamer

このリポジトリは、2024年9月にLaTeX Beamerを使用して作成した自己紹介スライドを含んでいます。

## ファイル一覧

- `LaTeX_sankaku.tex`: Beamerで作成したプレゼンテーションのLaTeXファイル
- `fig`: プレゼンテーションで使用している画像が格納されているファイル
- `LaTeX_sankaku.pdf`: コンパイル済みのPDFファイル
